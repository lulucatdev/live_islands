# Notice

LiveIslands is an independent project for React and Vue component islands in Phoenix LiveView.

The project was informed by and initially extracted from MIT-licensed work in these upstream projects:

- `mrdotb/live_react` by Mrdotb: https://github.com/mrdotb/live_react
- `Valian/live_vue` by Jakub Skalecki: https://github.com/Valian/live_vue

LiveIslands keeps this attribution visible because those projects established the practical shape of end-to-end client component islands for LiveView. The current project substantially rewrites and unifies the runtime around React and Vue adapters, shared LiveView contracts, Vite/Tailwind integration, SSR, lazy hydration strategies, and an install verifier workflow.
